Before any todo item is added, there is a button on the screen with a plus sign where the user can click to add a new item.
After clicking the add button and typing in the name of the task, the todo item appears on the screen, above the add button. The add button is pushed below because it makes it easier to reach by hand. The tasks are shown at the top to signify hierarchy.
