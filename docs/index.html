Task 1 Part 1: Before any todo item is added, there is a button on the screen with a plus sign where the user can click to add a new item.
Task 1 Part 2: After clicking the add button and typing in the name of the task, the todo item appears on the screen, above the add button. The add button is pushed below because it makes it easier to reach by hand. The tasks are shown at the top to signify hierarchy.
Task 2: After adding a new item to a list with one item.
Task 3 Part 1: A list with one item that is uncompleted.
Task 3 Part 2: A list with one item that is now completed. The completed item changes color and the text is decorated with line-through.
Task 4 Part 1: An editable item. HTML tag is given contenteditable='true' for the purposes of the wireframe.
Task 4 Part 2: Item after being edited.
Task 5 Part 1: The completed items are shown. The menu icon is clicked to show the option 'Hide Completed Items'.
Task 5 Part 2: After clicking the option 'Hide Completed Items', the completed items are now hidden, but they can be toggled to be shown again.
Task 6 Part 1: The completed items are shown. The menu icon is clicked to show the option 'Delete Completed Items'.
Task 6 Part 2: After clicking the option 'Delete Completed Items', the completed items are now permanently removed from the DOM.
